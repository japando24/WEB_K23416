CSS Position, Float, Overflow, Z-index

1. CSS Position (Vị Trí)

1.1 position: static (Mặc định)

position: static;  /* Mặc định - theo flow bình thường */
/* top, left, right, bottom KHÔNG có tác dụng */

1.2 position: relative

position: relative;
top: 20px;    /* dịch xuống 20px so với VỊ TRÍ GỐC */
left: 30px;   /* dịch phải 30px so với vị trí gốc */
/* Vẫn chiếm không gian ở vị trí gốc */
/* Quan trọng: dùng làm "gốc" cho absolute bên trong */
Phần tử trước
relative: dịch xuống 20px, phải 30px
Phần tử sau (vẫn ở vị trí cũ)

1.3 position: absolute

/* Container cha phải có position != static */
.parent {
    position: relative;  /* làm gốc tham chiếu */
}

.child {
    position: absolute;
    top: 10px;    /* cách top của cha 10px */
    right: 10px;  /* cách right của cha 10px */
}
/* THOÁT KHỎI flow - không chiếm không gian */
Đây là container (position: relative)
Phần tử absolute sẽ bám vào container này
ABSOLUTE (góc trên phải)

1.4 position: fixed

position: fixed;
top: 20px;    /* cách top CỬA SỔ TRÌNH DUYỆT 20px */
right: 20px;
/* Không cuộn theo trang - cố định với viewport */
/* Dùng cho: navbar, back-to-top button, chat widget */
fixed (demo với relative) - thực tế cố định với viewport

Ví dụ thực tế:

/* Nút Back to Top */
.back-to-top {
    position: fixed;
    bottom: 30px;
    right: 30px;
    background: #3498db;
    color: white;
    padding: 10px 15px;
    border-radius: 50%;
    cursor: pointer;
}

1.5 position: sticky

position: sticky;
top: 0;   /* Dính vào top=0 khi scroll đến đây */
/* Như relative bình thường, nhưng "dính" khi cuộn */
/* Dùng cho: sticky header, sticky sidebar */

Bảng So Sánh Position

Position Flow bình thường Tham chiếu top/left Cuộn trang
static ✓ Có N/A ✓ Cuộn
relative ✓ Có (giữ chỗ) Vị trí gốc của chính nó ✓ Cuộn
absolute ✗ Thoát flow Cha gần nhất có position ✓ Cuộn
fixed ✗ Thoát flow Viewport (cửa sổ trình duyệt) ✗ Cố định
sticky ✓ Có (giữ chỗ) Viewport khi scroll đến Hybrid

2. Z-index (Thứ Tự Xếp Chồng)

/* z-index chỉ hoạt động khi position != static */
.box1 { position: absolute; z-index: 1; }   /* dưới cùng */
.box2 { position: absolute; z-index: 2; }   /* giữa */
.box3 { position: absolute; z-index: 3; }   /* trên cùng */

/* Số LỚN HƠN = ở TRÊN */
/* Có thể dùng số âm */
z-index: 1
z-index: 2
z-index: 3 (trên)

3. CSS Overflow (Nội Dung Tràn)

overflow: visible;  /* MẶC ĐỊNH: tràn ra ngoài */
overflow: hidden;   /* Cắt bỏ nội dung tràn */
overflow: scroll;   /* Thanh cuộn luôn hiển thị */
overflow: auto;     /* Thanh cuộn khi cần thiết */

/* Từng chiều */
overflow-x: hidden;  /* ẩn tràn ngang */
overflow-y: scroll;  /* cuộn dọc */

visible (mặc định):

Nội dung dài hơn container. Tràn ra ngoài vì overflow: visible là mặc định. Text tiếp tục...

hidden:

Nội dung dài hơn container. Bị cắt khi overflow: hidden. Text tiếp tục...

scroll:

Nội dung dài hơn container. Có thanh cuộn luôn hiển thị dù không cần. Text tiếp tục...

auto:

Nội dung dài hơn container. Tự động thêm thanh cuộn khi cần. Text tiếp tục...

4. CSS Float (Nổi)

float: left;   /* Phần tử nổi sang trái, text bao quanh bên phải */
float: right;  /* Phần tử nổi sang phải, text bao quanh bên trái */
float: none;   /* Không nổi (mặc định) */

/* CLEAR: xóa ảnh hưởng float */
clear: both;   /* Không cho phần tử nằm cạnh float bên nào */
clear: left;   /* Không nằm cạnh float-left */
clear: right;  /* Không nằm cạnh float-right */

float: left

float: left

Nội dung text sẽ tự động bao quanh phần tử float left này. Bạn có thể thấy text chạy sang phải của hình.

Dòng text thứ hai cũng tiếp tục bao quanh.

float: right

float: right

Nội dung text sẽ tự động bao quanh phần tử float right này. Text chạy sang trái của hình.

Dòng text thứ hai cũng tiếp tục bao quanh.

Clearfix (xóa float ảnh hưởng)

/* Cách 1: Thêm phần tử rỗng */
<div style="clear: both;"></div>

/* Cách 2: overflow: hidden trên container */
.container { overflow: hidden; }

/* Cách 3: Clearfix hiện đại (dùng ::after) */
.clearfix::after {
    content: "";
    display: table;
    clear: both;
}

Tóm Tắt Quan Trọng

  • Position: static (mặc định), relative (dịch so với gốc), absolute (bám cha có position), fixed (bám viewport)
  • Absolute cần cha position: relative để tham chiếu đúng
  • Z-index chỉ hoạt động khi position != static
  • Z-index lớn hơn = ở trên
  • Overflow: visible, hidden, scroll, auto
  • Float: left, right → cần clear: both sau đó
