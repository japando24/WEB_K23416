Thuật ngữ Flexbox

Main Axis: Trục chính (hướng flex-direction)

Cross Axis: Trục phụ (vuông góc với main axis)

Flex Container: Phần tử cha có display: flex

Flex Items: Các phần tử con trực tiếp của container

3. Item Properties (Thuộc Tính Của Từng Item)

3.1 flex-grow (Mở rộng)

flex-grow: 0;   /* Không mở rộng (mặc định) */
flex-grow: 1;   /* Mở rộng chiếm 1 phần không gian còn lại */
flex-grow: 2;   /* Mở rộng chiếm 2 phần (gấp đôi flex-grow:1) */

/* Ví dụ: 3 items với grow 1, 2, 1 → chia 4 phần → item2 chiếm 2/4 */

flex-grow: 1, 2, 1 (chia theo tỉ lệ):

grow:1
grow:2 (gấp đôi)
grow:1

3.2 flex-shrink (Thu nhỏ khi không đủ chỗ)

flex-shrink: 0;   /* Không thu nhỏ */
flex-shrink: 1;   /* Thu nhỏ bình thường (mặc định) */
flex-shrink: 2;   /* Thu nhỏ nhiều hơn (gấp đôi) */

3.3 flex-basis (Kích thước ban đầu)

flex-basis: auto;   /* Theo kích thước nội dung (mặc định) */
flex-basis: 100px;  /* Kích thước cố định */
flex-basis: 50%;    /* Phần trăm của container */
flex-basis: 0;      /* Không có kích thước gốc (cộng với flex-grow) */

3.4 flex shorthand (QUAN TRỌNG!)

/* flex: grow shrink basis */
flex: 1;          /* = flex: 1 1 0 — phổ biến nhất! chia đều không gian */
flex: auto;       /* = flex: 1 1 auto */
flex: none;       /* = flex: 0 0 auto — không thay đổi kích thước */
flex: 2;          /* gấp đôi flex:1 */
flex: 0 0 200px;  /* cố định 200px */

3.5 order (Thứ tự hiển thị)

order: 0;    /* mặc định */
order: -1;   /* Hiển thị trước (thứ tự nhỏ hơn = trước) */
order: 1;    /* Sau order: 0 */
order: 2;    /* Cuối cùng */

HTML: 1,2,3 | CSS order: 3,1,2 → hiển thị: 1,2,3

align-self: flex-end;   /* Cuối cross axis */

Căn giữa hoàn toàn (Horizontal + Vertical)

.container {
    display: flex;
    justify-content: center;  /* Căn giữa ngang */
    align-items: center;      /* Căn giữa dọc */
    height: 300px;
}

flex-directionContainerrow* | row-reverse | column | column-reverse
flex-wrapContainernowrap* | wrap | wrap-reverse
justify-contentContainerflex-start* | flex-end | center | space-between | space-around | space-evenly
align-itemsContainerstretch* | flex-start | flex-end | center | baseline
align-contentContainerstretch* | flex-start | flex-end | center | space-between | space-around
flex-growItem0* | số dương
flex-shrinkItem1* | số không âm

flexItemflex: 1 (phổ biến nhất)
orderItem0* | số nguyên
