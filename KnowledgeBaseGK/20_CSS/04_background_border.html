8. Background Shorthand (Viết tắt)

/* background: color image repeat attachment position / size */
background: #3498db url('img.jpg') no-repeat fixed center / cover;

/* Ví dụ đơn giản */
background: red;
background: #fff url('img.jpg') no-repeat center;
background: linear-gradient(to right, red, blue);

10. CSS Icons

Bootstrap Icons

/* Trong <head>: */
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.10.0/font/bootstrap-icons.css">

/* Trong HTML: */
<i class="bi bi-heart"></i>
<i class="bi bi-star"></i>
<i class="bi bi-house"></i>

Google Material Icons

/* Trong <head>: */
<link href="https://fonts.googleapis.com/icon?family=Material+Icons" rel="stylesheet">

/* Trong HTML: */
<i class="material-icons">home</i>
<i class="material-icons">favorite</i>
<i class="material-icons" style="font-size: 48px; color: red;">star</i>

Font Awesome

/* Trong <head>: */
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.0.0/css/all.min.css">

/* Trong HTML: */
<i class="fas fa-home"></i>
<i class="fas fa-heart" style="color: red;"></i>
<i class="fab fa-github"></i>   /* fab = brand icons */

Tóm Tắt Quan Trọng

  • Background shorthand: background: color image repeat attachment position
  • background-size: cover (phủ toàn bộ), contain (không cắt)
  • background-repeat: repeat, no-repeat, repeat-x, repeat-y
  • border shorthand: border: width style color
  • border-radius: 50% → hình tròn
  • Styles: solid, dashed, dotted, double
