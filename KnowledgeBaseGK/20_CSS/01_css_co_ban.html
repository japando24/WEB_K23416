CSS Cơ Bản: Syntax, Types, Units, Comments

1. CSS Syntax (Cú pháp CSS)

selector {
    property: value;
    property: value;
}

/* Ví dụ */
h1 {
    color: red;
    font-size: 24px;
}

Thành phần:

  • selector: Chọn phần tử HTML cần style
  • property: Thuộc tính CSS (color, font-size, margin...)
  • value: Giá trị của thuộc tính

2. CSS Types (3 Loại CSS)

A. Inline CSS (CSS trực tiếp)

Viết trong thuộc tính style="" của thẻ HTML

<p style="color: red; font-size: 20px;">Văn bản đỏ</p>

Đây là Inline CSS (viết thẳng trong thẻ HTML)

B. Internal CSS (CSS nhúng)

Viết trong thẻ <style> trong <head>

<head>
  <style>
    p { color: blue; font-size: 16px; }
  </style>
</head>

C. External CSS (CSS ngoài)

Link đến file .css riêng

<head>
  <link rel="stylesheet" href="style.css">
</head>

Độ ưu tiên CSS (Priority)

Từ cao đến thấp:

  1. Inline style: style="..." (ưu tiên CAO NHẤT)
  2. ID selector: #id { }
  3. Class selector: .class { }
  4. Element selector: p { } (ưu tiên THẤP NHẤT)

Inline (crimson) thắng id (blue) thắng class (green) thắng element (orange)

3. CSS Comments (Chú thích CSS)

/* Đây là chú thích CSS - không hiển thị trên trình duyệt */

/* Chú thích 1 dòng */

/*
  Chú thích
  nhiều dòng
*/

p {
    color: red;  /* comment sau property */
    font-size: 16px;
}

Chú thích CSS bắt đầu bằng /* và kết thúc bằng */

4. CSS Units (Đơn vị CSS)

Absolute Units (Tuyệt đối)

font-size: 16px (pixels)

font-size: 12pt (points)

font-size: 0.5cm (centimeters)

font-size: 5mm (millimeters)

font-size: 1pc (pica)

Relative Units (Tương đối)

font-size: 120% (120% của cha)

font-size: 1.2em (1.2 lần font-size cha)

font-size: 1.2rem (1.2 lần font-size root)

Font-size Keywords

xx-small

x-small

small

medium (mặc định)

large

x-large

xx-large

larger (lớn hơn cha)

smaller (nhỏ hơn cha)

Bảng tóm tắt Units

Unit Loại Mô tả Ví dụ
px Tuyệt đối Pixels - phổ biến nhất 16px
pt Tuyệt đối Points (1pt = 1/72 inch) 12pt
cm/mm/in Tuyệt đối Centimeter, Millimeter, Inch 2cm
pc Tuyệt đối Pica (1pc = 12pt) 1pc
% Tương đối Phần trăm của phần tử cha 50%
em Tương đối Nhân với font-size của cha 1.5em
rem Tương đối Nhân với font-size của root (html) 1.2rem

5. Tóm Tắt - Quan Trọng Cho Thi

  • CSS Syntax: selector { property: value; }
  • 3 loại CSS: Inline (style=""), Internal (<style>), External (link href)
  • Độ ưu tiên: Inline > #id > .class > element
  • Comment: /* ... */
  • Units phổ biến: px, %, em, rem
