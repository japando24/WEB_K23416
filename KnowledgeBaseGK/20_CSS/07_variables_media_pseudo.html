CSS Variables, Math Functions & Media Queries

1. CSS Variables (Biến CSS)

Khai báo và Sử dụng

/* 1. Khai báo biến trong :root (toàn cục) */
:root {
    --primary-color: #3498db;
    --font-size: 16px;
    --spacing: 20px;
}

/* 2. Sử dụng biến với var() */
.button {
    background-color: var(--primary-color);
    font-size: var(--font-size);
    padding: var(--spacing);
}

/* 3. Giá trị dự phòng (fallback) */
.element {
    color: var(--undefined-var, red);  /* dùng red nếu biến không tồn tại */
}

/* 4. Biến cục bộ (chỉ trong 1 selector) */
.card {
    --card-color: orange;
    background: var(--card-color);
}

/* 5. Thay đổi biến bằng JavaScript */
document.documentElement.style.setProperty('--primary-color', 'red');

Sử dụng biến --primary-color và --secondary-color:

Card dùng biến CSS

Padding: var(--spacing-lg), border-radius: var(--border-radius), box-shadow: var(--box-shadow)

Fallback: var(--undefined-var, red) → màu đỏ vì biến không tồn tại

Ưu điểm của CSS Variables

  • Thay đổi một chỗ, áp dụng toàn bộ trang
  • Dễ tạo themes (dark mode, light mode)
  • Có thể thay đổi bằng JavaScript
  • Kế thừa theo DOM (scope)

Ví dụ Thực Tế: Dark/Light Theme

/* Light theme (mặc định) */
:root {
    --bg-color: #ffffff;
    --text-color: #333333;
    --border-color: #dddddd;
}

/* Dark theme */
[data-theme="dark"] {
    --bg-color: #1a1a2e;
    --text-color: #e0e0e0;
    --border-color: #555555;
}

body {
    background: var(--bg-color);
    color: var(--text-color);
}

/* Kích hoạt dark mode bằng JS: */
document.body.dataset.theme = 'dark';

2. CSS Math Functions (Hàm Toán Học)

2.1 calc() - Tính toán

/* calc(): Tính toán, KẾT HỢP các đơn vị khác nhau */
width: calc(100% - 40px);      /* 100% trừ 40px cố định */
width: calc(50% - 1rem);
height: calc(100vh - 80px);    /* chiều cao viewport trừ header */
font-size: calc(16px + 0.5vw); /* font responsive */
padding: calc(var(--spacing) * 2);  /* kết hợp với biến */

/* Toán tử: + - * / */
/* Lưu ý: * và / cần ít nhất 1 số không có đơn vị */
width: calc(100% / 3);         /* ✓ đúng */
width: calc(100% * 2);         /* ✓ đúng */
/* width: calc(100% * 50%);   -- ✗ sai (2 số có đơn vị) */
width: calc(100% - 40px) — chiều rộng bằng 100% trừ 40px
width: calc(50% - 20px)
width: calc(50% - 20px)

2.2 min() - Giá trị nhỏ nhất

/* min(a, b): Chọn giá trị NHỎ HƠN */
width: min(500px, 80%);
/* - Màn hình rộng: 80% < 500px → dùng 80% */
/* - Màn hình hẹp: 80% > 500px → dùng 500px */
/* → Đảm bảo phần tử KHÔNG LỚN HƠN 500px */

font-size: min(5vw, 36px); /* Text responsive, tối đa 36px */
min(500px, 80%) — Không bao giờ rộng hơn 500px

2.3 max() - Giá trị lớn nhất

/* max(a, b): Chọn giá trị LỚN HƠN */
width: max(200px, 50%);
/* - Màn hình hẹp: 50% < 200px → dùng 200px */
/* - Màn hình rộng: 50% > 200px → dùng 50% */
/* → Đảm bảo phần tử ÍT NHẤT 200px */

font-size: max(14px, 1vw); /* Text tối thiểu 14px */
max(200px, 50%) — Luôn tối thiểu 200px

2.4 clamp() - Giữ trong khoảng

/* clamp(min, preferred, max) */
font-size: clamp(14px, 3vw, 28px);
/* - Tối thiểu: 14px */
/* - Ưu tiên: 3vw (3% viewport width) */
/* - Tối đa: 28px */
/* → Font responsive, không quá nhỏ hay quá lớn */

width: clamp(300px, 50%, 800px);
clamp(14px, 3vw, 28px) — Thu nhỏ cửa sổ để thấy font thay đổi

So sánh nhanh

Hàm Cú pháp Kết quả Use case
calc()calc(100% - 40px)Tính toán kết hợpTrừ border, margin khỏi %
min()min(500px, 80%)Lấy giá trị nhỏ hơnGiới hạn tối đa
max()max(200px, 50%)Lấy giá trị lớn hơnĐảm bảo tối thiểu
clamp()clamp(14px, 3vw, 28px)Giữ trong khoảng min-maxResponsive font/size

3. CSS Media Queries (Responsive Design)

Cú pháp cơ bản

/* Cú pháp */
@media type and (condition) {
    /* CSS áp dụng khi điều kiện đúng */
}

/* Loại (media type) */
@media screen { }  /* Màn hình (phổ biến nhất) */
@media print  { }  /* Khi in */
@media all    { }  /* Tất cả */

/* Điều kiện phổ biến */
@media screen and (max-width: 768px) { }    /* Nhỏ hơn hoặc bằng 768px */
@media screen and (min-width: 768px) { }    /* Lớn hơn hoặc bằng 768px */
@media screen and (max-width: 1200px) and (min-width: 768px) { }  /* Khoảng giữa */

Breakpoints phổ biến

/* Mobile First Approach (ưu tiên mobile) */
/* Mặc định: Mobile (< 576px) */

@media screen and (min-width: 576px)  { /* Small devices - điện thoại lớn */ }
@media screen and (min-width: 768px)  { /* Medium - tablet */ }
@media screen and (min-width: 992px)  { /* Large - laptop */ }
@media screen and (min-width: 1200px) { /* XL - desktop */ }

/* ======================================================== */

/* Desktop First Approach */
/* Mặc định: Desktop */

@media screen and (max-width: 1200px) { /* < Desktop */ }
@media screen and (max-width: 992px)  { /* < Laptop */ }
@media screen and (max-width: 768px)  { /* < Tablet */ }
@media screen and (max-width: 576px)  { /* Mobile */ }
Thu nhỏ cửa sổ trình duyệt để thấy màu thay đổi:
🔴 Đỏ = Mobile (<768px) | 🟡 Vàng = Tablet (≥768px) | 🟢 Xanh = Desktop (≥1024px)

Ví dụ Responsive Navigation

/* Mobile: menu dọc */
nav ul {
    display: flex;
    flex-direction: column;
}

/* Desktop: menu ngang */
@media screen and (min-width: 768px) {
    nav ul {
        flex-direction: row;
    }
}

Ví dụ Responsive Grid

.grid {
    display: flex;
    flex-wrap: wrap;
    gap: 20px;
}

.card {
    flex: 1 1 100%;   /* Mobile: 1 cột */
}

@media screen and (min-width: 600px) {
    .card {
        flex: 1 1 calc(50% - 10px);  /* Tablet: 2 cột */
    }
}

@media screen and (min-width: 900px) {
    .card {
        flex: 1 1 calc(33.33% - 14px);  /* Desktop: 3 cột */
    }
}

Các điều kiện Media Query khác

/* Orientation (chiều màn hình) */
@media screen and (orientation: portrait)  { }  /* Dọc */
@media screen and (orientation: landscape) { }  /* Ngang */

/* Resolution (độ phân giải) */
@media screen and (min-resolution: 2dppx) { }  /* Màn hình Retina / HiDPI */

/* Kết hợp nhiều điều kiện */
@media screen and (min-width: 600px) and (max-width: 900px) { }

/* Hoặc (hoặc) */
@media screen and (max-width: 600px), screen and (min-width: 1200px) { }

/* Phủ định */
@media not screen and (max-width: 600px) { }

Tóm Tắt Quan Trọng

  • CSS Variables: Khai báo --tên: giá trị trong :root, dùng var(--tên)
  • calc(): Tính toán kết hợp đơn vị. VD: calc(100% - 40px)
  • min(): Lấy giá trị nhỏ nhất. VD: min(500px, 80%)
  • max(): Lấy giá trị lớn nhất. VD: max(200px, 50%)
  • Media Query: @media screen and (max-width: Xpx) { }
  • Mobile First: Viết CSS mặc định cho mobile, dùng min-width cho màn hình lớn hơn
  • Desktop First: Viết CSS mặc định cho desktop, dùng max-width cho màn hình nhỏ hơn
  • Breakpoints phổ biến: 576px (phone), 768px (tablet), 992px (laptop), 1200px (desktop)
