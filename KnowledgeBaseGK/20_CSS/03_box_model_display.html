CSS Box Model & Display

1. CSS Box Model (Mô Hình Hộp)

Mỗi phần tử HTML là một "hộp" với 4 lớp từ trong ra ngoài:

MARGIN (ngoài cùng)
BORDER (viền)
PADDING (đệm)
CONTENT (nội dung) 
.box {
    width: 200px;          /* chiều rộng CONTENT */
    height: 100px;         /* chiều cao CONTENT */
    padding: 20px;         /* khoảng đệm bên trong */
    border: 5px solid blue; /* viền */
    margin: 30px;          /* lề bên ngoài */
}

/* Tính tổng width:
   Total width = width + padding-left + padding-right + border-left + border-right
               = 200  + 20           + 20            + 5           + 5
               = 250px */
Content (200x100px)
Tổng width = 250px

box-sizing: content-box vs border-box

/* content-box (MẶC ĐỊNH): width = content only, padding/border CỘNG THÊM */
.box { box-sizing: content-box; width: 200px; padding: 20px; border: 5px solid; }
/* -> Tổng width thực tế = 250px */

/* border-box: width BAO GỒM padding + border */
.box { box-sizing: border-box; width: 200px; padding: 20px; border: 5px solid; }
/* -> Tổng width thực tế = 200px (không đổi) */

/* BEST PRACTICE: Thêm vào đầu CSS */
* { box-sizing: border-box; }
content-box: padding/border CỘNG vào width → rộng hơn 200px
border-box: padding/border NẰM TRONG width → vẫn 200px

2. Margin (Lề ngoài)

/* Một phía */
margin-top: 10px;
margin-right: 20px;
margin-bottom: 10px;
margin-left: 20px;

/* Shorthand */
margin: 20px;              /* 4 phía = 20px */
margin: 10px 30px;         /* top/bottom=10px, left/right=30px */
margin: 5px 15px 25px 35px; /* top right bottom left (theo kim đồng hồ) */
margin: 0 auto;            /* Căn giữa phần tử block! */
margin: 20px (4 phía)
margin: 10px 30px
margin: 0 auto (căn giữa)
Lưu ý: margin: 0 auto chỉ hoạt động khi phần tử có width cố định và là phần tử block

3. Padding (Khoảng đệm bên trong)

/* Một phía */
padding-top: 10px;
padding-right: 20px;
padding-bottom: 10px;
padding-left: 20px;

/* Shorthand (giống margin) */
padding: 20px;              /* 4 phía */
padding: 10px 30px;         /* top/bottom left/right */
padding: 5px 15px 25px 35px; /* top right bottom left */
padding: 5px (nhỏ)
padding: 20px (trung bình)
padding: 5px 40px (top/bottom nhỏ, left/right lớn)

So sánh Margin vs Padding

Đặc điểm Margin Padding
Vị trí Bên ngoài border Bên trong border
Ảnh hưởng background Không (trong suốt) Có (màu background phủ)
Giá trị âm Được phép (kéo phần tử) Không được phép
auto Dùng để căn giữa Không hỗ trợ auto

4. CSS Display

display: block

display: block;
/* - Chiếm toàn bộ chiều rộng (width: 100% mặc định)
   - Bắt đầu trên dòng mới (xuống dòng)
   - Nhận width và height
   - Ví dụ thẻ block: div, p, h1-h6, ul, li, section */

Block 2 (xuống dòng tự động)

display: inline

display: inline;
/* - Chỉ chiếm đúng kích thước nội dung
   - Không xuống dòng (nằm cùng hàng)
   - KHÔNG nhận width/height!
   - Ví dụ thẻ inline: span, a, strong, em, img */

/* - Nằm cùng hàng như inline
   - NHẬN width/height như block

5. Tóm Tắt Quan Trọng

  • Box Model: Content → Padding → Border → Margin (từ trong ra ngoài)
  • Total width = width + 2*padding + 2*border (với content-box)
  • box-sizing: border-box → width đã bao gồm padding và border
  • margin: 0 auto → căn giữa phần tử block
  • Margin shorthand: top right bottom left (theo kim đồng hồ)
  • display: block (xuống dòng, có w/h), inline (cùng hàng, không có w/h), inline-block (cùng hàng, có w/h)
  • display: none vs visibility: hidden
