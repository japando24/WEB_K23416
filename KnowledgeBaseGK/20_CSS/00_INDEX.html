💡 Ghi Nhớ Nhanh Cho Thi

  • Độ ưu tiên CSS: Inline > #id > .class > element
  • Box Model tổng width = width + 2×padding + 2×border (content-box)
  • Position absolute cần cha có position: relative
  • Flexbox căn giữa: display:flex; justify-content:center; align-items:center;
  • Media query: @media screen and (max-width: 768px) { }
  • CSS Variable: :root { --color: red; } → color: var(--color);
  • Child vs Descendant: parent > child (trực tiếp) vs parent child (tất cả)
  • float cần clear: both để reset
  • z-index chỉ hoạt động khi position != static
  • flex: 1 = flex: 1 1 0 (chia đều không gian)
