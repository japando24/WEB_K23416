CSS Selectors & Pseudo Classes/Elements

1. Các Loại Selector (Bộ Chọn)

1.1 Element Selector (Chọn theo thẻ)

p { color: red; }
h1 { font-size: 24px; }

Đây là thẻ <p> - element selector áp dụng cho TẤT CẢ thẻ p

1.2 #ID Selector (Chọn theo ID - duy nhất)

#my-id { color: blue; }   /* Trong HTML: id="my-id" */

Đây là phần tử có id="my-id-demo" — màu xanh

1.3 .Class Selector (Chọn theo class)

.my-class { color: green; }   /* Trong HTML: class="my-class" */

Phần tử 1 có class="my-class-demo"

Phần tử 2 cùng class - đều xanh lá

1.4 element.class (Chọn thẻ cụ thể có class)

p.highlight { color: orange; }   /* Chỉ <p class="highlight"> */

Đây là <p class="highlight"> - màu cam

Đây là <div class="highlight"> - KHÔNG bị ảnh hưởng bởi p.highlight

1.5 * Universal Selector (Chọn tất cả)

* { box-sizing: border-box; margin: 0; padding: 0; }

Thường dùng để reset CSS mặc định của browser

1.6 element, element (Nhóm selector)

h1, h2, p { color: purple; }   /* Áp dụng cho h1 VÀ h2 VÀ p */

Tiêu đề h4 - màu tím

Đoạn văn p - cũng màu tím (cùng nhóm selector)

2. Child vs Descendant Selector (QUAN TRỌNG!)

Child Selector: parent > child

.box > p { color: red; }
/* Chỉ chọn <p> là con TRỰC TIẾP */

Con trực tiếp - BỊ style (đỏ)

Cháu (p trong div) - KHÔNG bị style

Descendant Selector: parent descendant

.box p { color: teal; }
/* Chọn TẤT CẢ <p> bên trong */

Con trực tiếp - BỊ style (teal)

Cháu (p trong div) - CŨNG bị style

3. Pseudo Classes (Lớp giả)

Cú pháp: selector:pseudo-class { }

:hover - Khi rê chuột

button:hover { background: blue; color: white; }
Rê chuột vào đây!

a:link, a:visited, a:hover, a:active

a:link    { color: blue; }    /* Link chưa thăm */
a:visited { color: purple; }  /* Link đã thăm */
a:hover   { color: red; }     /* Khi hover */
a:active  { color: green; }   /* Đang click */
Link ví dụ - hover để xem đổi màu

:first-child và :last-child

p:first-child { color: red; }   /* Con đầu tiên */
p:last-child  { color: blue; }  /* Con cuối cùng */

Đoạn 1 - FIRST CHILD (đỏ)

Đoạn 2 - bình thường

Đoạn 3 - bình thường

Đoạn 1 - bình thường

Đoạn 2 - bình thường

Đoạn 3 - LAST CHILD (xanh)

:nth-child(n)

li:nth-child(2)    { color: red; }     /* Con thứ 2 */
li:nth-child(odd)  { background: #f0f; } /* Con lẻ */
li:nth-child(even) { background: #0ff; } /* Con chẵn */
  • Item 1 (lẻ - xanh lá)
  • Item 2 (chẵn + bold - vàng, được chọn đặc biệt)
  • Item 3 (lẻ - xanh lá)
  • Item 4 (chẵn - xanh dương)
  • Item 5 (lẻ - xanh lá)

4. Pseudo Elements (Phần tử giả)

Cú pháp: selector::pseudo-element { } (hai dấu ::)

::before và ::after

p::before { content: "⭐ "; }   /* Chèn TRƯỚC nội dung */
p::after  { content: " ✓"; }   /* Chèn SAU nội dung */

Đây là nội dung chính

Đây là nội dung chính

::first-letter (Chữ cái đầu tiên)

p::first-letter { font-size: 2em; color: red; float: left; }

Chữ đầu tiên của đoạn văn này sẽ được phóng to và đổi màu đỏ. Đây là kỹ thuật dropcap phổ biến trong thiết kế báo.

::first-line (Dòng đầu tiên)

p::first-line { color: blue; font-weight: bold; }

Dòng đầu tiên này sẽ có màu xanh và in đậm.
Các dòng tiếp theo sẽ có style bình thường.
Dòng thứ ba - không bị ảnh hưởng.

::selection (Khi bôi đen text)

p::selection { background: yellow; color: black; }

Bôi đen đoạn văn này để thấy màu selection vàng!
