05 - Fetch API: Đọc JSON từ File / API

1. So sánh XMLHttpRequest vs Fetch API

XMLHttpRequest (cũ) Fetch API (mới, hiện đại)
Cú pháp Dài, callback lồng nhau Gọn hơn, dùng Promise (.then / .catch)
Xử lý lỗi Kiểm tra readyState & status Kiểm tra response.ok, dùng .catch()
Async Có thể sync (false) hoặc async Luôn async (Promise)
Trong slide Dùng với XML Dùng với JSON

2. Template Fetch API đọc file JSON

// ====== TEMPLATE: FETCH JSON TỪ FILE ======

function loadJson() {
  var myList = document.querySelector('ul'); // Lấy element ul

  fetch("./data/products.json") // Gửi request GET đến file JSON
    .then(function(response) { // Nhận response
      if (!response.ok) { // Kiểm tra lỗi HTTP
        throw new Error("HTTP error, status = " + response.status);
      }
      return response.json(); // Parse response body thành JSON object
    })
    .then(function(json) { // Nhận JSON object đã parse
      // Xử lý dữ liệu JSON
      for (let p of json.products) { // json.products → Array
        var listItem = document.createElement('li');
        listItem.innerHTML = '<strong>' + p.id + '</strong>';
        listItem.innerHTML += p.name;
        listItem.innerHTML += ' --> <strong>' + p.price + '</strong>';
        myList.appendChild(listItem); // Thêm vào danh sách
      }
    })
    .catch(function(error) { // Bắt lỗi
      alert(error.message);
    });
}
Luồng Fetch API:
fetch(url) → .then(response => response.json()) → .then(json => xử lý) → .catch(error => xử lý lỗi)
