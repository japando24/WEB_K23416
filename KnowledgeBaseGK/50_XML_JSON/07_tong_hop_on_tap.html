Ôn Tập Giữa Kỳ: XML & JSON

PHẦN 1: XML

1.1 Khái niệm XML

XML (eXtensible Markup Language) = Ngôn ngữ đánh dấu "mở rộng", dùng để lưu trữ và truyền tải dữ liệu.
Khác HTML: XML tự định nghĩa thẻ, không hiển thị dữ liệu, chỉ mô tả dữ liệu.
4 quy tắc bắt buộc:
1. Thẻ mở phải có thẻ đóng: <name>...</name>
2. Phân biệt chữ HOA/thường: <Name> ≠ <name>
3. Phải có đúng 1 root element
4. Giá trị thuộc tính phải trong nháy kép/đơn
XML DOM = 3 thành phần:
Thành phầnVí dụMô tả
Element<name>Thẻ XML
Attributecategory="cooking"Thuộc tính của thẻ
TextHeinekenNội dung văn bản trong thẻ

1.2 Công thức lấy giá trị từ XML DOM

xmlDoc.getElementsByTagName("tên_thẻ")[chỉ_số].childNodes[0].nodeValue
// Parse chuỗi → XML Document
var parser = new DOMParser();
var xmlDoc = parser.parseFromString(xmlString, "text/xml");

// Lấy tất cả phần tử <product>
var products = xmlDoc.getElementsByTagName("product");

// Lấy text của <name> trong product đầu tiên
var ten = products[0].getElementsByTagName("name")[0].childNodes[0].nodeValue;

PHẦN 2: AJAX (XMLHttpRequest)

2.1 Khái niệm AJAX

AJAX = Asynchronous JavaScript And XML
Cho phép cập nhật trang web mà không cần tải lại, bằng cách gửi request HTTP từ JavaScript.

2.2 Template AJAX đầy đủ (5 bước)

// BƯỚC 1: Tạo đối tượng
var xhr = new XMLHttpRequest();

// BƯỚC 2: Cấu hình — open(method, url, async)
xhr.open("GET", "./data/products.xml", false);

// BƯỚC 3: Đăng ký callback
xhr.onreadystatechange = function() {
  if (xhr.readyState == 4 && xhr.status == 200) {
    // Thành công: xử lý dữ liệu
    var xmlDoc = xhr.responseXML; // Lấy XML doc
    var items = xmlDoc.getElementsByTagName("product");
    for (var i = 0; i < items.length; i++) {
      var name = items[i].getElementsByTagName("name")[0].childNodes[0].nodeValue;
      // Dùng name để build HTML...
    }
  } else {
    alert("Không lấy được dữ liệu");
  }
};

// BƯỚC 4: Gửi request
xhr.send();

2.3 Bảng thuộc tính quan trọng

Thuộc tínhGiá trị cần nhớ
readyState4 = Hoàn thành
status200 = OK; 404 = Not Found
responseXMLTrả về XML Document (dùng được DOM)
responseTextTrả về chuỗi text

PHẦN 3: JSON

3.1 Khái niệm JSON

JSON = JavaScript Object Notation. Định dạng key:value, phổ biến để trao đổi dữ liệu trên web.
JSON Object { }
{
  "id": 1,
  "name": "Heineken",
  "price": 19000
}
Truy cập: obj.name
JSON Array [ ]
[
  { "id": 1, ... },
  { "id": 2, ... }
]
Truy cập: arr[0].name

3.2 Hàm JSON quan trọng

HàmTác dụngVí dụ
JSON.parse(string) Chuỗi → Object JS JSON.parse('{"a":1}') → {a:1}
JSON.stringify(obj) Object JS → Chuỗi JSON.stringify({a:1}) → '{"a":1}'

3.3 Template Fetch API (đọc JSON từ file hoặc API)

fetch("url_hoac_duong_dan_file")
  .then(function(response) {
    if (!response.ok) { throw new Error("Error: " + response.status); }
    return response.json(); // Parse body → JSON object
  })
  .then(function(json) {
    // Trường hợp 1: json là Array → dùng trực tiếp
    for (let item of json) { ... }

    // Trường hợp 2: json là Object bọc ngoài → truy cập property
    for (let item of json.products) { ... }
  })
  .catch(function(error) {
    alert(error.message);
  });

PHẦN 4: So Sánh Tổng Hợp

Tiêu chí XML + AJAX (XHR) JSON + Fetch API
Đọc dữ liệu responseXML + getElementsByTagName response.json() → truy cập trực tiếp
Kiểm tra thành công readyState == 4 && status == 200 response.ok
Xử lý lỗi Trong else của if .catch(function(error) {...})
Cú pháp Phức tạp hơn, nhiều bước Gọn hơn, chuỗi .then()
Dùng trong slide Phần XML (AJAX) Phần JSON (Fetch)

PHẦN 5: Checklist Ôn Tập

  • XML là gì, khác HTML ở điểm nào
  • 4 quy tắc cú pháp XML phải nhớ
  • XML DOM: Element, Attribute, Text node
  • Cách dùng DOMParser để parse chuỗi XML
  • Công thức lấy nodeValue: getElementsByTagName("x")[0].childNodes[0].nodeValue
  • AJAX là gì, 5 bước thực hiện AJAX với XHR
  • readyState: 4 = xong | status: 200 = OK
  • responseXML vs responseText
  • JSON Object { } vs JSON Array [ ]
  • JSON.parse() và JSON.stringify()
  • Template Fetch API: fetch → .then(response.json()) → .then(data) → .catch()
  • Khi nào dùng json.products vs json trực tiếp
Ghi nhớ nhanh:
• XML + AJAX: dùng new XMLHttpRequest(), kiểm tra readyState==4 && status==200, đọc responseXML
• JSON + Fetch: dùng fetch(url).then(r => r.json()).then(data => ...)
