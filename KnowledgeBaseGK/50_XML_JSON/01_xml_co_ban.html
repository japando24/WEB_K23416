01 - XML Cơ Bản & XML DOM

1. Cấu trúc XML

XML (eXtensible Markup Language): Ngôn ngữ đánh dấu dùng để lưu trữ và truyền tải dữ liệu.
Khác HTML: XML tự định nghĩa thẻ, không hiển thị, chỉ mô tả dữ liệu.
<!-- Cấu trúc XML cơ bản -->
<?xml version="1.0" encoding="UTF-8"?> <!-- Dòng khai báo -->
<products> <!-- Root element (bắt buộc phải có 1 root) -->
  <product> <!-- Child element -->
    <id>1</id> <!-- Text node -->
    <name>Heineken</name>
    <price>19000</price>
  </product>
</products>
Lưu ý quan trọng:
  • Các thẻ mở phải có thẻ đóng tương ứng
  • Tên thẻ phân biệt chữ hoa/thường: <Name> ≠ <name>
  • XML document phải có đúng 1 thẻ root
  • Giá trị thuộc tính phải đặt trong nháy kép hoặc nháy đơn

2. XML DOM (Document Object Model)

XML DOM gồm 3 thành phần chính: Element (thẻ), Attribute (thuộc tính), Text (nội dung văn bản)
// ===== BƯỚC 1: Chuyển chuỗi XML thành XML Document =====
var xmlString = "<products><product>";
xmlString += "<id>1</id><name>Heineken</name><price>19000</price>";
xmlString += "</product></products>";

var parser = new DOMParser(); // Tạo parser
var xmlDoc = parser.parseFromString(xmlString, "text/xml"); // Parse chuỗi → XML Doc

// ===== BƯỚC 2: Truy cập phần tử =====
// Lấy theo tên thẻ → trả về HTMLCollection (mảng các phần tử)
var products = xmlDoc.getElementsByTagName("product"); // Lấy tất cả <product>

// Lấy text value của phần tử đầu tiên trong product đầu tiên
var tenSanPham = products[0].getElementsByTagName("name")[0] // thẻ <name> đầu tiên
                          .childNodes[0] // text node
                          .nodeValue; // lấy giá trị → "Heineken"
