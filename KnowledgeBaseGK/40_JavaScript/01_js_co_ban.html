⚡ 01 — JS Cơ Bản & Biến

JavaScript là ngôn ngữ lập trình client-side, tạo bởi Brendan Eich năm 1995. Chuẩn: ECMAScript (ES6, ES2021...).

1. Thẻ <script>

Vị trí đặt script

<!-- Đặt trong <head> (cần defer/async để không block HTML) --> <head> <script src="app.js" defer></script> </head> <!-- Đặt trước </body> (phổ biến nhất — HTML đã load xong) --> <body> ...nội dung... <script src="app.js"></script> </body> <!-- Inline (viết JS trực tiếp) --> <script> console.log("Hello World"); </script>

async vs defer

Thuộc tínhTải file JSChạy khi nàoDùng khi

2. Hiển Thị Dữ Liệu

// 1. innerHTML — thay nội dung HTML (phổ biến nhất) document.getElementById("myDiv").innerHTML = "<b>Hello!</b>"; // 2. innerText — thay nội dung TEXT (không parse HTML) document.getElementById("myDiv").innerText = "Hello <b>World</b>"; // hiển thị như text thô // 3. document.write() — ghi vào trang (chỉ dùng khi đang load!) document.write("<h1>Chào!</h1>"); // 4. window.alert() — hộp thoại thông báo alert("Bạn đã click!"); // hoặc window.alert() // 5. console.log() — in ra console (F12) console.log("Debug:", myVariable); // 6. window.print() — in trang window.print();

3. Cú Pháp JavaScript

// Câu lệnh (Statement) — kết thúc bằng ; (tuỳ chọn, nên có) let x = 5; let y = 10; let z = x + y; // z = 15 // Code block — dùng {} if (x > 0) { console.log("x dương"); // block của if } // Case-sensitive (phân biệt hoa thường) let myName = "An"; // ✅ let MyName = "Bình"; // khác biến trên! // Comment // Đây là comment 1 dòng /* Đây là comment nhiều dòng */ // Naming convention: lowerCamelCase let firstName = "An"; // ✅ biến let totalPrice = 100; // ✅ biến function calculateTax() {} // ✅ hàm const MAX_SIZE = 100; // ✅ hằng số (UPPER_SNAKE_CASE)

Từ Khóa (Keywords)

var, let, const // khai báo biến if, else // điều kiện switch, case // rẽ nhánh for, while // vòng lặp function, return // hàm try, catch, finally // xử lý lỗi class, new // class và object import, export // module break, continue // điều khiển vòng lặp

5. Kiểu Dữ Liệu (Data Types)

// Primitive types let num = 42; // Number let flt = 3.14; // Number (float) let str = "Hello"; // String let bool = true; // Boolean (true/false) let nul = null; // Null (không có giá trị) let undef = undefined; // Undefined (chưa gán) // Reference types let arr = [1, 2, 3]; // Array let obj = {name: "An"}; // Object let fn = function(){}; // Function // Kiểm tra kiểu typeof 42 // "number" typeof "hello" // "string" typeof true // "boolean" typeof null // "object" (quirk của JS!) typeof undefined // "undefined" typeof [] // "object" typeof {} // "object" typeof function(){} // "function" // Type conversion Number("42") // 42 Number("abc") // NaN String(42) // "42" Boolean(0) // false Boolean("") // false Boolean(null) // false Boolean("hi") // true Boolean(1) // true

📌 Tóm Tắt Quan Trọng

  • Đặt script trước </body> hoặc dùng defer trong head
  • Hiển thị: innerHTML (HTML), innerText (text), alert() (popup), console.log() (debug)
  • Dùng let cho biến thông thường, const cho hằng — tránh var
  • Block scope: let/const chỉ sống trong {} khai báo nó
  • Hoisting: var được đưa lên đầu (giá trị = undefined), let/const không dùng được trước khai báo
  • typeof null === "object" — đây là bug lịch sử của JS, cần nhớ
