PHẦN A — STRING (Chuỗi)

Khai báo và truy cập

let str = "Hello World"; str[0] // "H" (truy cập ký tự theo index) str.length // 11 (số ký tự) str[str.length - 1] // "d" (ký tự cuối)

PHẦN B — ARRAY (Mảng)

Khai báo

// Cách 1: array literal (phổ biến) const fruits = ["Táo", "Chuối", "Cam"]; const nums = [1, 2, 3, 4, 5]; const mixed = [1, "hello", true, null, {name: "An"}]; // mảng hỗn hợp // Cách 2: new Array() const arr = new Array(3); // [undefined, undefined, undefined] const arr2 = new Array(1,2,3); // [1, 2, 3] // Truy cập fruits[0] // "Táo" fruits[fruits.length - 1] // "Cam" (phần tử cuối) // Kiểm tra là mảng Array.isArray(fruits) // true Array.isArray("str") // false

Method thêm/xoá phần tử

let arr = ["a", "b", "c"]; // THÊM VÀO CUỐI — push() — trả về độ dài mới arr.push("d"); // arr = ["a","b","c","d"] arr.push("e", "f"); // thêm nhiều phần tử // XOÁ Ở CUỐI — pop() — trả về phần tử bị xoá let last = arr.pop(); // last = "f", arr = ["a","b","c","d","e"] // THÊM VÀO ĐẦU — unshift() — trả về độ dài mới arr.unshift("X"); // arr = ["X","a","b","c","d","e"] // XOÁ Ở ĐẦU — shift() — trả về phần tử bị xoá let first = arr.shift(); // first = "X", arr = ["a","b","c","d","e"] // THÊM/XOÁ Ở GIỮA — splice(index, deleteCount, ...newItems) arr.splice(2, 0, "Z"); // Thêm "Z" tại index 2: ["a","b","Z","c","d","e"] arr.splice(2, 1); // Xoá 1 phần tử tại index 2: ["a","b","c","d","e"] arr.splice(1, 2, "X","Y");// Xoá 2, thêm X,Y tại index 1: ["a","X","Y","d","e"] // NỐI MẢNG — concat() let a = [1, 2]; let b = [3, 4]; let c = a.concat(b); // [1, 2, 3, 4] (a,b không đổi) // CẮT MẢNG — slice(start, end) — không sửa mảng gốc let nums = [0,1,2,3,4,5]; nums.slice(2, 5); // [2, 3, 4] (từ index 2 đến trước 5) nums.slice(3); // [3, 4, 5] (từ index 3 đến cuối)

Method duyệt mảng (Functional)

const nums = [1, 2, 3, 4, 5]; // forEach — duyệt không return nums.forEach((n, i) => console.log(i, n)); // map — biến đổi, trả về mảng mới cùng độ dài const doubled = nums.map(n => n * 2); // [2, 4, 6, 8, 10] // filter — lọc, trả về mảng con const evens = nums.filter(n => n % 2 === 0); // [2, 4] const gt3 = nums.filter(n => n > 3); // [4, 5] // reduce — gộp thành 1 giá trị const sum = nums.reduce((acc, n) => acc + n, 0); // 15 const max = nums.reduce((acc, n) => n > acc ? n : acc, nums[0]); // 5 // find — tìm phần tử đầu tiên thoả điều kiện const found = nums.find(n => n > 3); // 4 (phần tử đầu tiên > 3) // findIndex — index của phần tử tìm được const idx = nums.findIndex(n => n > 3); // 3 // some — TỒN TẠI ít nhất 1 phần tử thoả điều kiện? nums.some(n => n > 4); // true // every — TẤT CẢ thoả điều kiện? nums.every(n => n > 0); // true nums.every(n => n > 3); // false // includes — có chứa giá trị không? nums.includes(3); // true nums.includes(9); // false // indexOf — vị trí đầu tiên (-1 nếu không có) nums.indexOf(3); // 2 nums.lastIndexOf(3); // 2

Sort, Reverse, ToString

const arr = [3, 1, 4, 1, 5, 9]; // sort() — sắp xếp mảng gốc (cẩn thận với số!) arr.sort(); // [1, 1, 3, 4, 5, 9] — đúng cho string! // Với số: sort mặc định so sánh string "10" < "9"! [10, 9, 100].sort() // [10, 100, 9] — SAI! [10, 9, 100].sort((a,b) => a-b) // [9, 10, 100] — ĐÚNG (tăng dần) [10, 9, 100].sort((a,b) => b-a) // [100, 10, 9] — giảm dần // reverse() — đảo ngược mảng gốc [1,2,3].reverse() // [3, 2, 1] // toString() — chuyển mảng sang chuỗi [1,2,3].toString() // "1,2,3" // join() — nối với ký tự phân cách tùy chọn [1,2,3].join(" - ") // "1 - 2 - 3" [1,2,3].join("") // "123" // Array.from() — tạo mảng từ iterable Array.from("Hello") // ["H","e","l","l","o"] Array.from({length: 5}, (_, i) => i) // [0, 1, 2, 3, 4] // Spread operator (ES6) — copy/nối mảng const copy = [...arr]; // copy mảng const merged = [...arr, ...copy]; // nối mảng

📌 Tóm Tắt Quan Trọng

  • String: slice(start, end) (âm OK) | substring (không âm) | substr(start, length)
  • Đảo chuỗi: "abc".split("").reverse().join("")
  • Array thêm/xoá: push/pop (cuối) | unshift/shift (đầu) | splice (giữa)
  • map trả về array mới | forEach không return | filter lọc phần tử
  • reduce gộp thành 1 giá trị (sum, max, group...)
  • Sort số: PHẢI dùng sort((a,b) => a-b) — sort mặc định sai với số!
  • slice không thay đổi mảng gốc | splice thay đổi mảng gốc
  • find → phần tử | findIndex → index | includes → boolean
