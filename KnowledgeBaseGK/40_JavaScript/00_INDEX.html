💡 Ghi Nhớ Nhanh Cho Thi JavaScript

  • var = function scope + hoisting | let/const = block scope, không hoisting
  • == so sánh giá trị (tự ép kiểu) | === so sánh giá trị VÀ kiểu
  • Arrow function: const fn = (x) => x * 2; — tự động return nếu không có {}
  • Array destructive: push/pop (đuôi) | unshift/shift (đầu) | splice (giữa)
  • forEach không return | map return array mới | filter lọc phần tử
  • DOM: document.getElementById("id") → 1 element | querySelectorAll → NodeList
  • Event: element.addEventListener("click", function)
  • parseInt("12px") → 12 | Number("12px") → NaN
  • Template literal: `Hello ${name}` thay vì cộng chuỗi
  • typeof kiểm tra kiểu: "number", "string", "boolean", "object", "undefined", "function"
