⚡ 07 — DOM & Events

1. DOM là gì? (Document Object Model)

DOM là cây đối tượng biểu diễn cấu trúc HTML. JavaScript dùng DOM để đọc và thay đổi nội dung, cấu trúc, style của trang.

document └── <html> ├── <head> │ ├── <title> "My Page" │ └── <meta charset="UTF-8"> └── <body> ├── <h1 id="title"> "Hello" ├── <p class="text"> "World" └── <ul> ├── <li> "Item 1" └── <li> "Item 2"

3. Thuộc Tính & Nội Dung Element

const el = document.getElementById("myDiv"); // Đọc / Ghi nội dung el.innerHTML = "<b>Bold</b>"; // parse HTML tags el.innerText = "Plain text <b>"; // hiển thị text thuần el.textContent = "Text content"; // text content (kể cả hidden) // Thuộc tính ID, tagName, className el.id // "myDiv" el.tagName // "DIV" el.className // "card active" el.classList // DOMTokenList — dùng add/remove/toggle/contains // classList el.classList.add("active"); // thêm class el.classList.remove("active"); // xoá class el.classList.toggle("active"); // toggle (thêm nếu chưa có, xoá nếu có) el.classList.contains("active"); // kiểm tra có class không → true/false // getAttribute / setAttribute / removeAttribute el.getAttribute("href") // lấy giá trị attribute el.setAttribute("href", "/home") // đặt attribute el.removeAttribute("disabled") // xoá attribute // Style trực tiếp el.style.color = "red"; el.style.fontSize = "18px"; el.style.backgroundColor = "#fff"; el.style.display = "none"; // ẩn phần tử // Node navigation el.parentNode // node cha el.parentElement // element cha el.children // HTMLCollection các con trực tiếp el.childNodes // NodeList gồm cả text node el.firstElementChild // con đầu tiên el.lastElementChild // con cuối cùng el.nextElementSibling // anh/chị tiếp theo el.previousElementSibling // anh/chị trước

4. Tạo, Thêm & Xoá Element

// Tạo element mới const newDiv = document.createElement("div"); const newLi = document.createElement("li"); const newText = document.createTextNode("Hello"); // Gán nội dung và thuộc tính newDiv.innerHTML = "<strong>Mới tạo!</strong>"; newDiv.className = "card"; newDiv.id = "newCard"; // Thêm vào DOM const container = document.getElementById("container"); container.appendChild(newDiv); // thêm vào cuối container.insertBefore(newDiv, container.firstChild); // thêm vào đầu // insertAdjacentHTML — linh hoạt hơn el.insertAdjacentHTML("beforeend", "<li>Item mới</li>"); // Positions: "beforebegin", "afterbegin", "beforeend", "afterend" // Xoá element const item = document.getElementById("itemToRemove"); item.remove(); // ES5+: xoá trực tiếp item.parentNode.removeChild(item); // cách cũ hơn // Clone element const copy = el.cloneNode(true); // true = clone cả children container.appendChild(copy);

3 cách gắn event

// CÁCH 1: HTML attribute (inline) — đơn giản nhưng không tách biệt <button onclick="myFunction()">Click</button> <input onchange="handleChange(this.value)"> // CÁCH 2: DOM property — ghi đè nếu gán nhiều lần const btn = document.getElementById("myBtn"); btn.onclick = function() { alert("Clicked!"); }; // Chỉ giữ handler cuối cùng: btn.onclick = handler1; // bị ghi đè bởi dòng dưới btn.onclick = handler2; // chỉ handler2 chạy // CÁCH 3: addEventListener — KHUYẾN KHÍCH NHẤT btn.addEventListener("click", function(event) { console.log("Clicked!", event); console.log(event.target); // element kích hoạt event console.log(event.target.id); // id của element đó }); // Gắn nhiều handler cho cùng 1 event btn.addEventListener("click", handler1); btn.addEventListener("click", handler2); // cả hai đều chạy! // Xoá event btn.removeEventListener("click", handler1); // Event object — thông tin về sự kiện btn.addEventListener("click", function(e) { e.preventDefault(); // ngăn hành vi mặc định (vd: submit form) e.stopPropagation(); // ngăn event bubble lên cha console.log(e.target); // element gốc kích hoạt console.log(e.currentTarget); // element đang xử lý console.log(e.type); // loại event: "click" console.log(e.clientX, e.clientY); // toạ độ chuột }); // Event Delegation — gắn event ở cha, xử lý từ con document.getElementById("list").addEventListener("click", function(e) { if (e.target.tagName === "LI") { console.log("Click vào:", e.target.textContent); } }); // DOMContentLoaded — đảm bảo HTML đã sẵn sàng document.addEventListener("DOMContentLoaded", function() { // code ở đây chạy sau khi DOM đã parse xong const el = document.getElementById("myEl"); // an toàn });

📌 Tóm Tắt Quan Trọng

  • getElementById → 1 element | querySelectorAll → NodeList tất cả
  • innerHTML parse HTML | innerText chỉ text | textContent text kể cả hidden
  • classList.add/remove/toggle/contains để quản lý class
  • Tạo element: createElement → setAttribute → appendChild
  • Xoá: element.remove() hoặc parent.removeChild(el)
  • addEventListener là cách tốt nhất — gắn được nhiều handler
  • event.preventDefault() ngăn hành vi mặc định (form submit, link...)
  • Event Delegation: gắn event ở cha, check e.target để xác định con
  • Wrap code trong DOMContentLoaded nếu script đặt trong head
