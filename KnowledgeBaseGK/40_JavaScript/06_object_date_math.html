PHẦN A — OBJECT (Đối Tượng)

Khai báo Object

// Object literal — cách phổ biến nhất const person = { name: "Nguyễn An", // property (thuộc tính) age: 20, city: "TP.HCM", isStudent: true, greet: function() { // method (phương thức) return "Xin chào, tôi là " + this.name; }, // ES6: method shorthand study() { return this.name + " đang học"; } }; // Arrow function làm method — KHÔNG dùng! (this sai) // bad: greet: () => "Hi " + this.name ← this không phải person!

Truy cập thuộc tính

const person = { name: "An", age: 20, "full name": "Nguyễn An" }; // Dot notation — phổ biến person.name // "An" person.age // 20 // Bracket notation — dùng khi tên có khoảng trắng hoặc dùng biến person["age"] // 20 person["full name"] // "Nguyễn An" let key = "name"; person[key] // "An" (dùng biến làm key) // Thêm / sửa / xoá property person.email = "an@example.com"; // thêm mới person.age = 21; // cập nhật delete person.city; // xoá // Kiểm tra property tồn tại "name" in person // true person.hasOwnProperty("age") // true // Lấy tất cả key / value Object.keys(person) // ["name", "age", "email"] Object.values(person) // ["An", 21, "an@example.com"] Object.entries(person) // [["name","An"], ["age",21], ...]

Destructuring và Spread

const person = { name: "An", age: 20, city: "HCM" }; // Destructuring — lấy nhanh nhiều property const { name, age } = person; console.log(name, age); // "An" 20 // Đổi tên khi destructure const { name: fullName, age: tuoi } = person; console.log(fullName, tuoi); // "An" 20 // Default value const { country = "Việt Nam" } = person; console.log(country); // "Việt Nam" // Spread — copy / merge object const copy = { ...person }; // copy const updated = { ...person, age: 21 }; // merge (ghi đè age) const merged = { ...obj1, ...obj2 }; // merge 2 object

Class (ES6)

// Định nghĩa class class Student { // constructor chạy khi new Student(...) constructor(name, age, gpa) { this.name = name; // gán thuộc tính this.age = age; this.gpa = gpa; } // Method greet() { return `Tôi là ${this.name}, GPA: ${this.gpa}`; } getGrade() { if (this.gpa >= 3.6) return "Xuất sắc"; if (this.gpa >= 3.2) return "Giỏi"; if (this.gpa >= 2.5) return "Khá"; return "Trung bình"; } } // Tạo instance (đối tượng từ class) const sv1 = new Student("Nguyễn An", 20, 3.7); const sv2 = new Student("Trần Bình", 21, 2.8); console.log(sv1.greet()); // "Tôi là Nguyễn An, GPA: 3.7" console.log(sv1.getGrade()); // "Xuất sắc" console.log(sv2.getGrade()); // "Khá" // Kế thừa (Inheritance) class GradStudent extends Student { constructor(name, age, gpa, major) { super(name, age, gpa); // gọi constructor cha this.major = major; } info() { return `${this.name} — Chuyên ngành: ${this.major}`; } }

PHẦN B — DATE OBJECT

// Tạo Date object const now = new Date(); // thời gian hiện tại const d1 = new Date("2024-01-15"); // từ string const d2 = new Date(2024, 0, 15, 10, 30, 0); // year, month(0-11), day, h, m, s const d3 = new Date(1705305600000); // từ timestamp (ms) // GET methods — đọc giá trị now.getFullYear() // Năm: 2026 now.getMonth() // Tháng: 0-11 (⚠️ 0=Tháng 1!) now.getMonth() + 1 // Tháng thực: 1-12 now.getDate() // Ngày trong tháng: 1-31 now.getDay() // Thứ trong tuần: 0=CN, 1=T2, ..., 6=T7 now.getHours() // Giờ: 0-23 now.getMinutes() // Phút: 0-59 now.getSeconds() // Giây: 0-59 now.getMilliseconds() // Milliseconds: 0-999 now.getTime() // Timestamp (ms từ 1/1/1970) // SET methods — đặt giá trị const d = new Date(); d.setFullYear(2025); d.setMonth(11); // Tháng 12 (index 11) d.setDate(25); // Ngày 25 // Format hiển thị now.toString() // "Sat Jun 28 2026 10:30:00 GMT+0700" now.toLocaleDateString() // "28/6/2026" (phụ thuộc locale) now.toLocaleTimeString() // "10:30:00 SA" now.toISOString() // "2026-06-28T03:30:00.000Z" now.toDateString() // "Sat Jun 28 2026" // Tính khoảng cách giữa 2 ngày const d_start = new Date("2024-01-01"); const d_end = new Date("2024-12-31"); const diff_ms = d_end - d_start; // hiệu số milliseconds const diff_days = diff_ms / (1000*60*60*24); // chuyển ra ngày

📌 Tóm Tắt Quan Trọng

  • Object: { key: value } — dot notation obj.key hoặc bracket obj["key"]
  • Object.keys() → mảng keys | Object.values() → mảng values | Object.entries() → [[k,v]...]
  • Class: constructor chạy khi new | kế thừa: extends + super()
  • Date: getMonth() = 0-11 → cần +1 để ra tháng thực!
  • getDay() = 0 (Chủ Nhật) đến 6 (Thứ Bảy)
  • Math.random() [0,1) → số nguyên [1,n]: Math.floor(Math.random()*n) + 1
  • Min/Max từ mảng: Math.min(...array) | Math.max(...array)
