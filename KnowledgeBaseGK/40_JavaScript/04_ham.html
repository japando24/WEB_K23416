⚡ 04 — Hàm (Function)

1. Function Declaration (Khai Báo Hàm)

// Cú pháp cơ bản function tênHàm(thamSố1, thamSố2) { // code return kếtQuả; // return là tuỳ chọn } // Ví dụ function greet(name) { return "Xin chào, " + name + "!"; } console.log(greet("An")); // "Xin chào, An!" // Hàm có nhiều tham số và giá trị mặc định (ES6) function power(base, exponent = 2) { // mặc định exponent = 2 return Math.pow(base, exponent); } console.log(power(3)); // 9 (3^2) console.log(power(2, 8)); // 256 (2^8) // Hoisting — function declaration được hoisted! console.log(add(2, 3)); // 5 — gọi TRƯỚC khi khai báo vẫn được! function add(a, b) { return a + b; } // Hàm không return — trả về undefined function sayHi(name) { console.log("Hi " + name); // không có return } let result = sayHi("Bình"); console.log(result); // undefined

2. Function Expression (Biểu Thức Hàm)

// Gán hàm vào biến const multiply = function(a, b) { return a * b; }; console.log(multiply(3, 4)); // 12 // ⚠️ Function expression KHÔNG được hoisted! // console.log(multiply(2,3)); // LỖI nếu gọi trước khai báo // Named function expression const factorial = function fact(n) { if (n <= 1) return 1; return n * fact(n - 1); // gọi đệ quy bằng tên }; console.log(factorial(5)); // 120

3. Arrow Function (Hàm Mũi Tên — ES6)

// Cú pháp đầy đủ const add = (a, b) => { return a + b; }; // Rút gọn: nếu body chỉ có 1 biểu thức → bỏ {} và return const add2 = (a, b) => a + b; // return ngầm định const square = x => x * x; // 1 tham số: bỏ () const greet = () => "Hello World!"; // không tham số: giữ () console.log(add2(3, 4)); // 7 console.log(square(5)); // 25 console.log(greet()); // "Hello World!" // Return object literal — phải bọc trong () const makePerson = (name, age) => ({ name, age }); // ← dùng () console.log(makePerson("An", 20)); // {name: "An", age: 20} // Arrow function dùng nhiều với array methods const nums = [1, 2, 3, 4, 5]; const doubled = nums.map(n => n * 2); // [2, 4, 6, 8, 10] const evens = nums.filter(n => n % 2 === 0); // [2, 4] const sum = nums.reduce((acc, n) => acc + n, 0); // 15
⚠️ Arrow function không có this riêng — this của arrow function = this của scope bên ngoài. Không dùng arrow function làm method của object.

4. Callback Function

// Callback = hàm được truyền vào hàm khác như tham số function doMath(a, b, operation) { return operation(a, b); // gọi callback } // Truyền hàm vào function add(a, b) { return a + b; } function multiply(a, b) { return a * b; } console.log(doMath(5, 3, add)); // 8 console.log(doMath(5, 3, multiply)); // 15 // Truyền anonymous function (hàm vô danh) console.log(doMath(5, 3, function(a, b) { return a - b; })); // 2 // Truyền arrow function console.log(doMath(5, 3, (a, b) => a ** b)); // 125 // Ví dụ phổ biến: addEventListener document.getElementById("btn").addEventListener("click", function() { console.log("Đã click!"); // đây là callback }); // Array methods dùng callback const names = ["Châu", "An", "Bình"]; names.forEach(function(name, index) { // callback nhận (item, index) console.log(index + ": " + name); });

5. Hàm Built-in Quan Trọng

Chuyển đổi số

// Number() — chuyển sang number (strict) Number("42") // 42 Number("3.14") // 3.14 Number("") // 0 Number("abc") // NaN Number(true) // 1 Number(false) // 0 Number(null) // 0 Number(undefined) // NaN // parseInt() — chuyển sang số nguyên (dừng khi gặp ký tự không hợp lệ) parseInt("42px") // 42 (lấy phần đầu!) parseInt("3.9") // 3 (cắt phần thập phân) parseInt("abc") // NaN parseInt("0xFF", 16) // 255 (parse hex) parseInt("10", 2) // 2 (parse binary) // parseFloat() — chuyển sang số thực parseFloat("3.14px") // 3.14 parseFloat("3.9") // 3.9 parseFloat("abc") // NaN // isNaN() — kiểm tra có phải NaN không isNaN("abc") // true (không phải số) isNaN(42) // false isNaN("42") // false ("42" ép được sang số) isNaN(NaN) // true Number.isNaN(NaN) // true (chặt hơn — không ép kiểu)

setTimeout và setInterval

// setTimeout — chạy 1 lần sau X milliseconds const timerId = setTimeout(function() { console.log("Chạy sau 2 giây!"); }, 2000); // 2000ms = 2 giây // Dùng arrow function setTimeout(() => alert("Hello sau 3 giây!"), 3000); // clearTimeout — huỷ trước khi chạy clearTimeout(timerId); // setInterval — chạy lặp lại mỗi X milliseconds let count = 0; const intervalId = setInterval(() => { count++; console.log("Lần " + count); if (count >= 5) { clearInterval(intervalId); // dừng sau 5 lần } }, 1000); // mỗi 1 giây // clearInterval — dừng setInterval clearInterval(intervalId);

📌 Tóm Tắt Quan Trọng

  • Function declaration được hoisted — có thể gọi trước khi khai báo
  • Arrow function rút gọn: const fn = x => x * 2 (return ngầm định)
  • Callback = hàm truyền vào hàm khác, rất phổ biến trong event, array methods
  • parseInt("12px") = 12 | Number("12px") = NaN
  • setTimeout(fn, ms) = 1 lần sau ms | setInterval(fn, ms) = lặp mỗi ms
  • Luôn lưu ID để clearTimeout(id) / clearInterval(id) khi cần dừng
