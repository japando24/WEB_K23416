3. Toán Tử Nối Chuỗi

// Cộng chuỗi với chuỗi let str1 = "Hello"; let str2 = "World"; let result = str1 + " " + str2; // "Hello World" // Cộng chuỗi với số — tự động ép kiểu! "5" + 3 // "53" (string concat!) "5" - 3 // 2 (số học — JS ép về number) "5" * 3 // 15 "5" + 3 + 2 // "532" (từ trái sang phải: "5"+3="53", "53"+2="532") 5 + 3 + "2" // "82" (5+3=8, 8+"2"="82") // Template Literal (ES6+) — dùng backtick ` let name = "An"; let age = 20; let msg = `Tên: ${name}, Tuổi: ${age}`; // "Tên: An, Tuổi: 20" let calc = `2 + 2 = ${2 + 2}`; // "2 + 2 = 4"
⚠️ Lưu ý: "5" + 3 = "53" (string!), nhưng "5" - 3 = 2 (number). Phép + với string sẽ nối chuỗi, các phép còn lại ép về number.

5. Toán Tử Logic

let age = 20; let hasID = true; // AND if (age >= 18 && hasID) { console.log("Được vào"); // ✅ cả hai đúng } // OR if (age < 18 || !hasID) { console.log("Không được vào"); } // NOT let isLoggedIn = false; if (!isLoggedIn) { console.log("Vui lòng đăng nhập"); } // Short-circuit evaluation // && — nếu vế trái false, không xét vế phải // || — nếu vế trái true, không xét vế phải let user = null; let name = user && user.name; // null (không crash dù user.name sẽ báo lỗi) let display = user || "Guest"; // "Guest" (default value pattern)

6. Toán Tử 3 Ngôi (Ternary)

// Cú pháp: điều_kiện ? giá_trị_nếu_đúng : giá_trị_nếu_sai let age = 20; let access = (age >= 18) ? "Được phép" : "Không được phép"; console.log(access); // "Được phép" // Thay thế if/else ngắn gọn let score = 75; let grade = (score >= 90) ? "A" : (score >= 80) ? "B" : (score >= 70) ? "C" : "F"; console.log(grade); // "C" // Dùng trong JSX/template let isOnline = true; let status = `Trạng thái: ${isOnline ? "🟢 Online" : "🔴 Offline"}`; console.log(status); // "Trạng thái: 🟢 Online"

📌 Tóm Tắt Quan Trọng

  • == (loose) tự ép kiểu — TRÁNH dùng! | === (strict) an toàn — NÊN dùng
  • Nối chuỗi: "a" + "b" hoặc template literal `Hello ${name}`
  • Ternary: condition ? valueA : valueB — thay thế if/else đơn giản
  • Short-circuit: user || "Guest" (default) | user && user.name (safe access)
  • % (modulo) dùng kiểm tra chẵn/lẻ: n % 2 === 0
  • ** là luỹ thừa: 2 ** 8 = 256
