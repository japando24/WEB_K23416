📋 Template 02 – select · datalist · textarea · fieldset · output

File mẫu có chú thích đầy đủ – Môn Phát triển Web Kinh doanh · UEL

1 · <select> Dropdown cơ bản <select>

💡 value là thứ server nhận, text bên trong là thứ user thấy. Luôn thêm option trống đầu tiên làm placeholder.

2 · <select> Multiple & Size multiple size

💡 multiple + size hiển thị list box. Người dùng giữ Ctrl/Cmd để chọn nhiều mục.

3 · <optgroup> Nhóm tùy chọn <optgroup>

💡 optgroup nhóm các option lại cho dễ nhìn. disabled vô hiệu hóa cả nhóm.

4 · <datalist> Gợi ý autocomplete <datalist>

💡 list= của input phải khớp với id= của datalist. Người dùng vẫn có thể gõ giá trị không có trong gợi ý.

5 · <textarea> Nhập nhiều dòng <textarea>

💡 rows/cols kiểm soát kích thước. Nội dung mặc định đặt giữa thẻ, không dùng value=.

6 · <fieldset> + <legend> Nhóm controls <fieldset>
🔐 Thông tin đăng nhập
📧 Thông tin liên hệ
7 · <output> Hiển thị kết quả <output>
0 100 + = 100

💡 oninput trên <form> chạy mỗi khi bất kỳ input nào thay đổi. output hiển thị kết quả ngay lập tức.
