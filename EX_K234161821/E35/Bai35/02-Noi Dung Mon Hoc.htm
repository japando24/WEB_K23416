Nội Dung Môn Học – Lập Trình Web

Môn học Lập Trình Web cung cấp kiến thức nền tảng về thiết kế và xây dựng trang web, bao gồm các công nghệ XHTML, CSS, JavaScript và DHTML. Sinh viên sau khi hoàn thành môn học có khả năng xây dựng website tĩnh và động cơ bản theo chuẩn W3C.

Thông Tin Môn Học

Tên môn học: Lập Trình Web
Số tín chỉ: 3 tín chỉ (2 lý thuyết + 1 thực hành)
Đối tượng: Sinh viên năm 2 ngành Công nghệ Thông tin
Học kỳ: Học kỳ I, năm học 2017 – 2018

Nội Dung Chi Tiết

  1. Phần 1: XHTML và CSS
    1. XHTML – Thẻ cơ bản, Danh sách, Liên kết
      • Cấu trúc tài liệu XHTML chuẩn
      • Các thẻ định dạng văn bản
      • Tạo danh sách có thứ tự và không thứ tự
      • Tạo và quản lý liên kết
    2. XHTML – Hình ảnh và Bảng biểu
      • Chèn và căn chỉnh hình ảnh
      • Tạo bảng dữ liệu với <table>, <tr>, <td>
      • Gộp ô, định dạng bảng
      • Ứng dụng bảng trong bố cục trang
    3. CSS – Cascading Style Sheets
      • Inline Style, Internal Style, External Style
      • Selector: thẻ, class, id
      • Thuộc tính: font, màu sắc, nền, lề, viền
      • Mô hình hộp (Box Model)
  2. Phần 2: Lập Trình JavaScript Cơ Bản
    1. JavaScript – Giới thiệu và cú pháp cơ bản
      • Lịch sử và vai trò của JavaScript
      • Biến, kiểu dữ liệu, toán tử
      • Nhập/xuất dữ liệu: alert, prompt, confirm
    2. JavaScript – Cấu trúc điều khiển (Phần 1)
      • Cấu trúc if – else if – else
      • Cấu trúc switch – case
      • Vòng lặp for
    3. JavaScript – Cấu trúc điều khiển (Phần 2)
      • Vòng lặp while, do – while
      • Lệnh break và continue
      • Bài tập tổng hợp cấu trúc điều khiển
    4. JavaScript – Hàm (Function)
      • Khai báo và gọi hàm
      • Tham số và giá trị trả về
      • Hàm ẩn danh (Anonymous Function)
    5. JavaScript – Mảng (Array)
      • Khai báo và truy xuất phần tử mảng
      • Các phương thức: push, pop, sort, splice
    6. JavaScript – Đối tượng (Object)
      • Khái niệm đối tượng và thuộc tính
      • Đối tượng có sẵn: Math, Date, String
  3. Phần 3: DHTML
    1. DHTML – Mô hình Đối Tượng và Collections
      • Document Object Model (DOM)
      • Truy xuất phần tử qua getElementById, getElementsByTagName
    2. DHTML – Mô hình Sự Kiện
      • Các sự kiện chuột, bàn phím
      • Xử lý sự kiện bằng JavaScript
    3. DHTML – Filters và Transitions
      • Hiệu ứng lọc màu sắc, đổ bóng
      • Hiệu ứng chuyển cảnh (Transitions)
    4. DHTML – DataBinding
      • Liên kết dữ liệu với phần tử HTML
      • Sử dụng Tabular Data Control (TDC)
